# ProsParity Collection Dashboard

This project is a Vite + React + TypeScript app using Tailwind CSS and shadcn/ui components.

## Local development

```sh
# Install dependencies
npm i

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
